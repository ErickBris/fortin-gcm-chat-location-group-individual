A) Introduction & Basic Configuration - top

Fortin Chat APP Using GCM Application is Native Android based Application which has backend as PHP.

Basic Configuration:

As soon as you download the code base from Codecaynon, please make the following changes to the server side php code.
  • Browse to the download folder > FortinGCMChat > Chatserver, copy paste this folder into the working directory of your application folder i.e. 'www' for LAMP/WAMP or 'htdocs' for XMPP Server.
  • Create database and run script with name chatdatabase.sql in phpmyadmin
  • Now you can change GOOGle API key in config.php
  • Note : we do not have any admin panel for this

B) PHP Files Structure - top

Following Image shows the basic folder structure of the application with PHP files.

 

Description of the PHP files as follows, Please go through the same if you wish to go for risking and changing the application as per your need.

db.php - Database connection file, please add all the required parameters like database name, user name and password.

GCM.php - Confrugation and functions to check valid user and send push notifications.

config.php -Define Google API key which is available on Google Developer Console.

Following are the webservices

createnewgroup.phpWebservice to create new Group with multiple members from Android app.

getGroupInfo.php Webservice to get information about like groupname, admin and member details.

getMyFriends.php Webservice to get My Friends. Once you send any individual message to any user, he/she will get automatically add in to your friends list.

getallusers.php Webservice to get all users registered with this app. Right now we are not using this webservice.

getnearbyfriends.phpWebservice to get all users in mentioned distance. We are sending longitude, latitude and distance for radius to get user from Android. This webservice will provide list of users in that area.

getuserprofile.phpWebservice to get selected user profile.

logout.php Webservice to logout from the server.

register.phpWebservice to register on Server using facebook and google+.

sendGroupMessage.phpWebservice to send group message using groupid.

sendIndividualMessage.phpWebservice to send individual message using userid.

updatelocation.phpWebservice to update longitude and latitude.

updatestatus.php webservice to update Status.

blockuser.php Webservice to Block user so user will not be able to send message each other.

deleteGroup.php Webservice to delete Group only by Admin of the Group.

UnBlockUser.php Webservice to Unblock user to start conversation again.

fileUpload.php Webservice to store files like audio, image and video on server.

getBlockedUser.php Webservice to all the blocked user for loggedin user.

updategroup.php Webservice to update Group name and add more users to group.

resetPassword.php Please place your domain email id to send email for current password
